Capabilities

Built for transparency

No black boxes. Every selection decision is logged, scored, and returned alongside the response.

🧠
Intent classification
Identifies primary and secondary intents across 9 categories with confidence scoring, so routing decisions are never arbitrary.
🛡
Pre-inference security
Detects prompt injection, PII exposure, jailbreak patterns, and policy violations before any model sees the query.
💰
Cost optimisation
Estimates token usage before inference and always picks the cheapest model that meets the complexity requirement.
📐
Complexity scoring
Scores every query from 1 to 10 with structured reasoning, ensuring simple chat never gets routed to an expensive expert model.

Enterprise AI Gateway
